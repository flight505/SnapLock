SnapLock

Creates parametric twist-snap bayonet containers (Lid + Receiver) optimized for FDM 3D printing. One smooth rotation locks the lid onto the container using inward tabs and angled slots.

Quick Start

  1. Click Solid → 3D Print Tools → SnapLock
  2. Set outer diameter, body height, and rim dimensions
  3. Adjust wall thicknesses (walls, rim, floor, cap)
  4. Set tab count and lock geometry (tab angle, entry angle)
  5. Choose what to create: Lid, Receiver, or Both
  6. Click OK

Size

  • Outer Diameter — total container diameter
  • Body Height — height of the receiver (without rim)
  • Rim Height — where the tabs engage

Lock Mechanism

  • Tabs — inward protrusions on the lid that engage the receiver slots
  • Tab Revolve Angle — arc length of each tab
  • Slot Entry Angle — how steep the entry ramp is
  • Column Protrusion — vertical column depth on the lid

FDM Tips

  • Walls 1.5–2mm for small containers, 2–3mm for larger
  • Print receiver upright (open end up), lid upside-down
  • 0.15–0.2mm layer height for clean tab geometry
  • Test fit: print a small section first to tune tolerances
